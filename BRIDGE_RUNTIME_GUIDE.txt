ALNOUR OFFERS - BRIDGE RUNTIME GUIDE
====================================

RUNTIME MODES
-------------
1) local
   - frontend -> local API
   - use:
     VITE_RUNTIME_MODE=local

2) bridge
   - frontend -> pharmacy bridge
   - use:
     VITE_RUNTIME_MODE=bridge
     VITE_BRIDGE_BASE_URL=http://PHARMACY-IP:5192
     VITE_BRIDGE_KEY=YOUR_BRIDGE_KEY

3) production
   - frontend -> production api
   - use:
     VITE_RUNTIME_MODE=production
     VITE_API_BASE_URL=https://api.example.com

HOW TO SWITCH TO BRIDGE
-----------------------
1) copy .env.bridge.example into .env
2) update:
   - VITE_BRIDGE_BASE_URL
   - VITE_BRIDGE_KEY
3) run:
   npm run dev

NOTES
-----
- In bridge mode, products are loaded from:
  /api/shamel/products

- Bridge mode currently treats reviews as unavailable unless exposed by a separate API.
