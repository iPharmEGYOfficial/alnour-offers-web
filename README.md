# AlNour Offers Web

واجهة متجر عروض صيدلية النور مبنية باستخدام React + Vite.

## الميزات

- الصفحة الرئيسية والهوية البصرية
- صفحة المنتجات والعروض
- تفاصيل المنتج
- السلة
- إتمام الطلب
- الفاتورة والطباعة
- الحساب والعناوين
- صفحات الإدارة الأساسية
- دعم Runtime محلي / Bridge / Production

## المتطلبات

- Node.js 18+
- npm 9+

## التشغيل المحلي

```bash
npm install
npm run dev
```
